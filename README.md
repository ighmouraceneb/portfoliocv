### Contribuer

Pour installer faites simplement les commandes suivantes:


```bash
  	$ npm install
  	$ bower install
```

et lancez

```bash
	$ gulp 
```

La tache par defaut installe les fichiers de styles, fonts et scripts puis surveille les sources pour injecter les modifications apportées.
